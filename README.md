# Onyx AI
<img width="1062" height="433" alt="Screenshot 2026-03-01 at 7 47 34 PM" src="https://github.com/user-attachments/assets/fd3477db-76bf-4f40-acfd-b3d2d5e28f40">

The AI for all computing learners. Built as a bedrock for all your computing knowledge, talk to Onyx to further your knowledge / studies in computing.

This is an ongoing project which is a Retrieval-Augmented Generation (RAG) based chatbot where Onyx is an AI conversational agent that enhances its responses by retrieving relevant, up-to-date information from external, trusted knowledge sources before generating an answer. In this case it retrieves chunks of information from a database of documents contributed by former H2 computing students and university students. This ensures that the answers provided by Onyx are specialised in contrast to the generic responses provided by chatbots like OpenAI, Gemini, etc..
